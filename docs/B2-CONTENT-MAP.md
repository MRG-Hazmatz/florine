# B2 Content Map — Alter Ego B2 → Florine b2.1 / b2.2

**Source:** *Alter Ego B2* (Hachette FLE) — themed-dossier edition, 9 dossiers.
**⚠ The PDF is a SCANNED book (no text layer).** Extraction workflow: PyMuPDF
`page.get_pixmap(dpi=200+).save(png)` → Read the PNG visually. `pdftoppm` and
direct PDF Reads are unavailable in this environment. PDF index ≈ book page − 2.

**Mapping:** each dossier → 2 units. **18 units total.**
- **b2.1** = Dossiers 1–5 (10 units)
- **b2.2** = Dossiers 6–9 (8 units)

Same conventions as B1 (see B1-CONTENT-MAP.md): original French (book = scope/
sequence blueprint only), ~18-exercise pool incl 2 speaking drills + 5 generated
pool drills, sampled + shuffled at runtime, every unit `pending_review`,
2 modules/turn at the established quality bar with a self-audit before commit.

Status: ✅ built · 🔨 in progress · ⬜ planned

## Notes
- B2 grammar is mostly **synthesis + refinement** of B1 systems, plus genuinely new
  items: passé simple/antérieur (recognition), ne explétif, adjectif verbal,
  registres de langue. Units should spiral back explicitly.
- Themes mature with the level: work, health, civic life, aesthetics, society.
- DELF B2 target: argumentation everywhere (the exam is argue-or-die).
- Activity Book B2 PDF (scanned, 106pp) available for extra exercise inspiration.
