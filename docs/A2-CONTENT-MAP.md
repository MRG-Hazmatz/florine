# A2 Content Map — Alter Ego+ 2 → Florine a2.1 / a2.2

**Source:** *Alter Ego+ 2* (Hachette FLE). Its avant-propos states it targets **A2.2 →
B1.1**, assuming **A2.1** on entry. That A2.1 entry grammar (passé composé, imparfait,
COD, partitifs, comparatif, en/y) is exactly what we already folded into **a1.2** from
the A1 book — so the learner arrives ready for Dossier 1 here.

**Mapping (Dossiers 1–4 = the book's A2.2 = Florine's "A2"):**
- **a2.1** = Dossiers 1–2 (J'ai des relations! · Vers la vie active)
- **a2.2** = Dossiers 3–4 (La France vue par… · Médiamania)
- **Dossiers 5–8** → **b1.1 / b1.2** later (the book places the DELF A2 exam at the end
  of D5/D6, marking the entry into B1).

Content is original; the book is used only as the scope/sequence blueprint, never copied.
Every module = objectif pragmatique + Point Langue grammar + thematic lexicon +
DELF-format exercises + native audio, self-audited, and `pending_review` until sign-off.
