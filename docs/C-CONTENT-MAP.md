# C1/C2 Content Map — Alter Ego 5 (C1>C2) → Florine c1.1 / c1.2 / c2.1 / c2.2

**Source:** *Alter Ego 5 C1>C2* (Hachette FLE) — 12 dossiers in 3 parties, with
built-in **DALF C1 entraînements** (parts 1–2) and **DALF C2 entraînements** (part 3).
**⚠ Scanned book, no text layer** — same workflow as B2: `fitz` pixmap → PNG → Read.

**Level mapping** (follows where the book places its own DALF entraînements):
- **c1.1** = Partie 1 *Changements*, Dossiers 1–4 → 8 units
- **c1.2** = Partie 2 *Nouveaux enjeux*, Dossiers 5–8 → 8 units
- **c2.1** = Partie 3 *Rêves*, Dossiers 9–10 → 4 units
- **c2.2** = Partie 3 *Rêves*, Dossiers 11–12 → 4 units
**24 units total.** C2 levels are deliberately smaller (the book's own C2 share);
size-to-content over symmetry.

**C-level unit design** — the book has no Point Langue: C is about *discourse
techniques* and high-register lexicon, not new grammar. Each dossier → 2 units:
one centred on the dossier's **technique** (synthèse, exposé, résumé, essai,
lettre administrative, dissertation, éditorial, revue de presse, critique,
rapport, roman), one on its **theme + language tools** (connectors, register,
figures, collocations). Concept/vocab/exercise format unchanged — concepts teach
the technique's method + discourse markers; exercises drill structure, register
and comprehension at C level.

## Notes
- DALF C1: synthèse + essai argumenté are THE exam tasks — units 1, 15 (synthèse)
  and 13 (dissertation) carry the heaviest exam weight.
- DALF C2 has only two épreuves (oral & written, each comprehension+production
  combined) — c2 units train long-form production above all.
- Audio: edge-tts remains fine for C-level listening drills (clear standard French);
  documents are longer — listen transcripts may run 2–3 sentences.
